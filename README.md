# 👨‍💼 Employee Management System (Java)

## 📌 Project Overview
This project is a console-based Employee Management System developed using Java.  
It allows users to manage employee records efficiently by performing operations such as adding, viewing, updating, and deleting employee details through a menu-driven interface.

---

## 🚀 Features
- ➕ Add new employee details (ID, Name, Salary)  
- 📋 View all employee records  
- ✏️ Update existing employee information  
- ❌ Delete employee records safely  
- ⚠️ Error handling for invalid inputs  
- 🧾 Menu-driven console application  

---

## 🛠️ Technologies Used
- Java  
- Object-Oriented Programming (OOP)  
- ArrayList (Dynamic Data Storage)  
- Scanner Class (User Input Handling)  

---

## 🧠 Concepts Covered
- Classes and Objects  
- Encapsulation  
- Constructors  
- ArrayList usage  
- CRUD Operations (Create, Read, Update, Delete)  
- Loops and Conditional Statements  
- Iterator (for safe deletion)
